# Math formulas
## Area
- Circle: S = πR²
**(площадь круша)**
- Rectangle: S = ab
**(площадь прямоугольника)**
- Square: S = a²
**(площадь квадрата)**

## Perimeter
- Circle: P = 2πR
**(длина окружности)**
- Rectangle: P = 2a + 2b
**(периметр прямоугольника)**
- Square: P = 4a
**(периметр квадрата)**
