# Math formulas, realised in lib
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle: S = ah/2

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2 * (a + b)
- Square: P = 4a
- Triangle: P = a + b + c

# Functions with examples
## Circle with R = 10
- S = 3,14 * 10^2 = 314 
- P = 2 * 3,14 * 10 = 62,8

## Rectangle with a = 5, b = 4
- S = a * b = 20
- R = 2 * (a + b) = 18

## Square with a = 7
- S = 7^2 = 49
- P = 4 * 7 = 28

# Triangle with a = 2, b = 3, c = 4 and h(a) = 8
- S = 2 * 8 / 2 = 8
- P = 2 + 3 + 4 = 9
